# Mystery Number Game

A simple number guessing game built with HTML, CSS, and JavaScript.

## How to Play

1. Guess a number between 1 and 10
2. Click "Make a Guess" or press Enter
3. Get feedback: too high, too low, or correct!
4. Find the secret number to win

## Game Features

- Random number generation (1-10)
- Color-coded feedback messages
- Detective theme styling
- Works on desktop and mobile
